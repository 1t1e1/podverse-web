How do I edit my clips?

In order to edit a clip, you must have been logged in when you created the clip. After you create the clip, open it on the website or the iOS mobile app, and there will be an Edit Clip button on the right hand side of the page.

Web: Go to the clip's page, then click Edit Clip.

iOS: open the clip in the media player, then tap Edit Clip.

How do I find my clips?

If you are not logged in when you create a clip, then you must save the link to the clip if you want to find it later.

If you are logged in when you create a clip, then you can find it in your My Clips list at any time on web or iOS.

Web: Go to the Home page, click the dropdown in the upperleft corner, then select My Clips.

iOS: Go to the Clips page, click the dropdown in the upperleft corner, then select My Clips.

What does AGPLv3 mean?

AGPLv3 is the open source license under which all Podverse technolgy is provided.

How is Podverse anti-monopoly?

Podverse software is distributed under the AGPLv3 open source license, which allows anyone to freely use, modify, and distribute the software, as long as they likewise share their own modifications to the code. The share-alike requirement of the AGPLv3 ensures that the code is and will always be free.

Also, to prevent data lock-in, future releases of Podverse will be built in accordance with podcast open data standards, so that all Podverse data will be easily compatible with any app that follows those same data standards. The benefit of this is, say you wanted to stop using Podverse and use a different podcast app instead, you could download all your Podverse data and transfer it into the competitor's app, and get up and running without losing anything in the process.
